# 한탭 차단 앱 — 설치 방법

앱 코드는 다 만들어졌어요. 이제 이걸 **APK(설치 파일)로 굽고 → 폰에 깔면** 끝입니다.
컴퓨터에 뭘 설치할 필요 없이, 무료 GitHub에서 자동으로 구워집니다.

---

## A. APK 굽기 (컴퓨터에서, 약 10분)

### 1) GitHub 계정 만들기
- https://github.com 접속 → Sign up → 이메일로 무료 가입.

### 2) 새 저장소(repository) 만들기
- 오른쪽 위 **+** → **New repository**
- Repository name: `call-block` (아무거나)
- **Private** 선택해도 됨 → **Create repository**

### 3) 앱 파일 올리기
- 만들어진 저장소 화면에서 **uploading an existing file** 글자 클릭
  (또는 Add file → Upload files)
- 이 폴더 안의 **`전화차단앱` 폴더 내용물 전체**를 끌어다 놓기:
  `app` 폴더, `.github` 폴더, `build.gradle`, `settings.gradle`, `gradle.properties`
  → ⚠️ `.github` 폴더가 꼭 같이 올라가야 자동 빌드가 됩니다.
  → ⚠️ `desktop.ini` 라는 파일이 보이면 **올리지 마세요**(구글 드라이브가 만든 쓰레기 파일, 빌드 방해 가능).
- 아래 **Commit changes** 버튼 클릭.

### 4) 자동 빌드 기다리기
- 위쪽 메뉴 **Actions** 탭 클릭 → "APK 빌드"가 노란 점(진행 중) → 초록 체크(완료)
- 3~5분 걸립니다. (빨간 X면 아래 'C. 막히면' 참고)

### 5) APK 내려받기
- 완료된 빌드를 클릭 → 맨 아래 **Artifacts**의 **한탭차단-앱** 클릭 → zip 다운로드
- 압축 풀면 `app-debug.apk` 파일이 나옵니다. 이걸 폰으로 보내세요
  (카톡 나에게 보내기 / 메일 / USB 등).

---

## B. 폰에 설치 (갤럭시)

1. 폰에서 `app-debug.apk` 파일을 탭 → "출처를 알 수 없는 앱" 경고가 뜨면
   **이 출처 허용**으로 설치 진행.
2. 설치 후 **한탭 차단** 앱 열기.
3. **① 전화 차단 기능 켜기** 버튼 누르기 → "이 앱을 발신자 표시·스팸 앱으로
   사용하시겠습니까?" 비슷한 창이 뜨면 **허용/예**.
4. 전화 권한·알림 권한 물어보면 모두 **허용**.

끝! 이제 이렇게 동작합니다:
- 전화가 오고 통화가 끝나면 → 알림에 **"방금 통화: 010-xxxx [차단]"** 이 뜸
- **차단**을 한 번 누르면 → 그 번호는 다음부터 자동으로 거절(무음)
- 그 번호 **문자는 그대로** 옴 (삼성 메시지에 빨간점 그대로)
- 앱을 열면 차단된 번호 목록이 보이고, **해제**로 풀 수 있음

---

## C. 막히면 (빌드 빨간 X)
- Actions에서 빨간 X 빌드를 누르면 어디서 멈췄는지 글로 나옵니다.
  그 화면을 캡처해서 저에게 주세요. 바로 고쳐드립니다.
